🚀 Advanced Features

Natural command execution with semantic search and safety-first automation

🔍 Interro-Powered Semantic Search

Find files and commands on your system using natural language. Powered by interro, it understands meaning, not just filenames or keywords.

📦 Local GGUF Model Support

Load and run open-source LLMs locally via GGUF format. Full support for Hugging Face-hosted models—no cloud dependency, full privacy.

🕓 History-Aware Shell Commands

Generate shell commands based on your previous command history and intent. Ollash learns from what you've done to help you do it faster.

🛡️ Reliable Multi-Layer Safety

Built-in command validation, risk assessment, and secure confirmation prompts protect you from harmful or destructive operations before they run.
